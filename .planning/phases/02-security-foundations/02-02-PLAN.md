  <action>
    1. **Create Enterprise/Audit/AuditLogger.h**: Follow Pattern 2 from research exactly:
       - `struct AuditEvent` with fields: timestamp, server, topic, username, database, clientIp, authMethod, text
       - `class AuditLogger` with:
         - `enum class OutputType { File, Syslog }`
         - `void addOutput(OutputType type, std::string const& target)` -- configure output destinations
         - `void log(AuditEvent event)` -- non-blocking enqueue to ring buffer
         - `void start()` -- launch background writer thread
         - `void stop()` -- flush remaining events, join writer thread
       - Private: `std::mutex _mutex`, `std::condition_variable _cv`, `std::deque<AuditEvent> _buffer`, `std::atomic<bool> _running`, `std::thread _writerThread`
       - Private: `void drainLoop()` -- background thread that drains buffer to all outputs

    2. **Implement Enterprise/Audit/AuditLogger.cpp**:
       - `log()`: Lock mutex, push_back event to deque, notify_one on CV. This is O(1) amortized. Do NOT write to file/syslog here.
       - `drainLoop()`: Loop while `_running` or buffer non-empty. Wait on CV with timeout (100ms). Swap buffer contents under lock (minimize lock hold time). Write each event to all configured outputs. Format: `"timestamp|server|topic|username|database|clientIp|authMethod|text\n"`
       - File output: `std::ofstream` opened in `addOutput()`, write formatted line, flush periodically (not per-event for performance)
       - Syslog output: `openlog("arangodb", LOG_PID, LOG_LOCAL0)` in addOutput, `syslog(LOG_INFO, "%s", formatted.c_str())` per event
       - `stop()`: Set `_running = false`, notify CV, join thread. The drainLoop continues until buffer is empty (Pitfall 5 mitigation).

    3. **Create tests/unit/AuditFeatureTest.cpp**: Tests per behavior block. For file output tests, write to a temp file and read back. For concurrent tests, spawn 8 threads each logging 100 events, verify all 800 appear in output. Skip syslog tests (requires system daemon) -- test that addOutput(Syslog, ...) doesn't crash.

    4. **Update CMakeLists.txt**: Add `Enterprise/Audit/AuditLogger.cpp` to `openarangodb_enterprise` sources.

    5. **Update tests/CMakeLists.txt**: Add `audit_feature_test` executable.
